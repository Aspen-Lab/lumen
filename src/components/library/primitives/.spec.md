# Lumen Primitives — Design Specification

## Shared Rules

Every primitive:
- Accepts `className` prop for external overrides
- Uses `style` prop for color props (not Tailwind classes) so colors are runtime-adjustable
- Has a default dark-on-dark appearance that works without any props
- Exports a named TypeScript interface for its props
- Self-contained — zero external dependencies beyond React + Framer Motion

---

## 01 — GlowButton

The signature Lumen button. Gradient border glow + press interaction.

```
Props:
  label: string
  sublabel?: string
  variant: "primary" | "ghost" | "danger"
  size: "sm" | "md" | "lg"
  glowFrom?: string          // gradient start  (#7C5CFC)
  glowTo?: string            // gradient end    (#F97316)
  glowIntensity?: number     // 0-1
  loading?: boolean
  disabled?: boolean
  icon?: ReactNode
  onClick?: () => void
```

Visual:
- primary: gradient bg matching glow, white text
- ghost: transparent bg, subtle border, text matches glowFrom
- danger: red tones
- Press: scale(0.97), shadow shrinks
- Hover: glow intensifies, slight lift
- Loading: spinner replaces label

---

## 02 — GhostButton

Minimal borderless button for secondary actions.

```
Props:
  label: string
  size: "sm" | "md"
  color?: string              // text/icon color
  icon?: ReactNode
  onClick?: () => void
```

Visual:
- No background at rest
- Hover: bg white/4%, text brightens
- Active: scale(0.97)

---

## 03 — Toggle

On/off switch.

```
Props:
  value: boolean
  onChange: (v: boolean) => void
  size: "sm" | "md"
  activeColor?: string        // track color when on
  label?: string
```

Visual:
- Track: rounded pill, white/8% off → activeColor/30% on
- Thumb: white circle, slides with spring
- Label optional, placed left

---

## 04 — Slider

Range input with value display.

```
Props:
  value: number
  onChange: (v: number) => void
  min: number
  max: number
  step?: number
  label?: string
  showValue?: boolean
  trackColor?: string
  thumbColor?: string
```

Visual:
- 2px track, filled portion uses trackColor
- 12px circle thumb
- Value readout in mono right-aligned

---

## 05 — Badge

Status/category label.

```
Props:
  label: string
  variant: "default" | "success" | "warning" | "danger" | "info"
  size: "sm" | "md"
  color?: string              // overrides variant color
  dot?: boolean               // show status dot before label
  mono?: boolean              // use mono font
```

Visual:
- Rounded pill, bg at 8% of color, text at 80% of color
- Optional pulsing dot
- sm: 10px text, md: 12px text

---

## 06 — StatusDot

Minimal state indicator.

```
Props:
  state: "active" | "complete" | "pending" | "error"
  size: "sm" | "md" | "lg"
  pulse?: boolean
  color?: string
```

Visual:
- Circle with optional pulse animation ring
- active: green, complete: white, pending: white/20, error: red

---

## 07 — ProgressBar

Linear progress indicator.

```
Props:
  value: number               // 0-1
  animated?: boolean
  color?: string
  trackColor?: string
  height?: number             // px, default 4
  rounded?: boolean
```

Visual:
- Track: subtle bg
- Fill: color, optionally spring-animated from 0
- Rounded or square ends

---

## 08 — ArcGauge

Semicircular gauge like ConfidenceMeter uses.

```
Props:
  value: number               // 0-1
  size: number                // px width
  strokeWidth?: number
  color?: string
  trackColor?: string
  showLabel?: boolean         // center percentage
  animated?: boolean
```

Visual:
- SVG arc ~200° sweep
- Animated fill with spring
- Center percentage text
- Optional glow on high values

---

## 09 — TextInput

Styled text input / textarea.

```
Props:
  value: string
  onChange: (v: string) => void
  placeholder?: string
  multiline?: boolean
  rows?: number
  bgColor?: string
  textColor?: string
  placeholderColor?: string
```

Visual:
- Dark bg, no visible border at rest
- Focus: subtle glow/brighten
- Auto-resize when multiline

---

## 10 — Tooltip

Hover/focus information popup.

```
Props:
  content: string | ReactNode
  side: "top" | "bottom" | "left" | "right"
  children: ReactNode
```

Visual:
- Dark elevated bg, small text
- Arrow pointing to trigger
- Fade+scale entry animation

---

## 11 — Popover

Click-triggered floating panel (used by ColorPicker etc).

```
Props:
  trigger: ReactNode
  content: ReactNode
  side: "top" | "bottom"
  align: "start" | "center" | "end"
  open?: boolean
  onOpenChange?: (open: boolean) => void
```

Visual:
- Elevated bg, shadow
- Click outside to close
- Scale+fade entry

---

## 12 — Tag

Inline label for categorization.

```
Props:
  label: string
  color?: string
  removable?: boolean
  onRemove?: () => void
```

Visual:
- Small rounded rect, bg at 5%, text at 50%
- Optional × remove button
